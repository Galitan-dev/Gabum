## `gabum upgrade`

upgrade the package to the last version

```
USAGE
  $ gabum upgrade [-w <value>] [-f]

FLAGS
  -f, --force
  -w, --with=<value>  package manager to use

DESCRIPTION
  upgrade the package to the last version

ALIASES
  $ gabum update
  $ gabum up

EXAMPLES
  $ gabum upgrade
```
