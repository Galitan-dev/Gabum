## `gabum use [VERSION]`

Checks for a previously installed version of the gabum CLI. Throws an error if the version is not found.

```
USAGE
  $ gabum use [VERSION]

ARGUMENTS
  VERSION  Specify an explicit version (ex. 3.0.0-next.1) or a channel (ex. alpha)

DESCRIPTION
  Checks for a previously installed version of the gabum CLI. Throws an error if the version is not found.
```
